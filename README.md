# PaN-Finder Project Deliverables
This repository contains the deliverables of the OSCARS PaN-Finder project and their current status.
The original project page on the OSCARS site can be found at this URL https://oscars-project.eu/projects/pan-finder-photon-and-neutron-federated-knowledge-finder.  
The current project application is accessible at https://pan-finder.panosc.ess.eu/


## List of Deliverables 

1.	Body of knowledge

    1.a.A repository with one or more files containing the body of knowledge in the selected format.  
        https://github.com/PaN-Finder/deliverables/tree/main/task-1/data
	
        Status: Done

	1.b	A code repository with all the scripts used to collect data and create the body of knowledge.  
        https://github.com/PaN-Finder/deliverables/tree/main/task-1/scripts  
        https://github.com/PaN-Finder/deliverables/tree/main/task-1/scripts-v1   

        Status: Done

2. Components selection	

	2.a	Report on exploration and activities for MVP.  
        (Formerly: A document containing the metrics collected, the selected components, models and parameters, the justification for the selection).  
        https://github.com/PaN-Finder/deliverables/blob/main/task-2/PaN-Finder-Deliverable-2-a.pdf  
	
        Status: Done

	2.b	Report on current system architecture.  
        (Formerly: A document highlighting the architecture for the RAG system and the LLM models with a set of examples).  
  
        Status: In progress

3. Main components deployment	

	3.a	Deployment of the RAG and LLMs services on ESS IT infrastructure with dedicated REST APIs accessible locally.  
       
        Status: Done

	3.b	Report benchmarks and test pairs.
        A document containing the tests results
	
        Status: In progress

4. System integration
	
	4.a	Deployment of a single REST API to interact with the system

        Status: Done

	4.b	Produce public documentation including examples

        Status: To be done

	4.c	Code repository with the code and relevant configuration

        Status: In Progress

5. User Interface
	
	5.a	Deployment of the new PaNOSC Data Portal with the PaN-Finder tool
        https://pan-finder.panosc.ess.eu/
        
        Status: Done

	5.b	New release of the PaNOSC Data Portal in the public code repository
	
        Status: To be done

6. Update strategy
	
	6.a	Production deployment of the update service on ESS infrastructure

        Status: To be done

	6.b	Code repository containing the update service

        Status: To be done

7. Community engagement
	
	7.a	Project summary document

        Status: To be done

	7.b	Presentations and Users Interviews
        (Formerly: Two community meetings/workshops)

        Status: In progress

	7.c	One presentation or a poster

        Status: To be done
